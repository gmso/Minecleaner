# Minecleaner
A clone of the classic Minesweeper, made for fun and learning purposes.

Makes use of the c++ SFML library

## Requirements

Windows 10 64 bit

## Usage

Execute the _.exe_ file, no installation needed (it's a very simple game ;) )
